# Neon-Nexus: Futuristic E-Commerce Starter
A high-performance, dark-mode tech store built for the 2026 electronics niche.

## Tech Stack
- **Frontend:** React.js (v18+) with Inline CSS-in-JS for zero-dependency styling.
- **Data Layer:** Scalable JSON-based product architecture (optimized for 2026 GA4 tracking).
- **Deployment:** Azure Static Web Apps with automated CI/CD via GitHub Actions.

## Key Features
- **Cyberpunk UI:** High-contrast neon interface with glassmorphism effects.
- **Dynamic Search:** Real-time filtering across 20+ futuristic product categories.
- **Low Overhead:** Hosted on Azure's Free Tier, meaning $0/month maintenance costs.
- **SEO Ready:** Structured for high-intent tech keywords like "Quantum Computing" and "Neural Wearables."
